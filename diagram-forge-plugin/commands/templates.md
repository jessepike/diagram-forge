# List Templates

Call the `list_templates` MCP tool and display the results.

## Steps

1. Call `list_templates` to get all available diagram types.

2. Display each template with:
   - Name and display name
   - Description
   - Supported diagram types
   - Required variables

3. Suggest the user try `/diagram:create` with their chosen template type.
